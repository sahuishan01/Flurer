// PRIORITY ISSUES
P1

No visible focus rings

outline: none is set on all form controls without a replacement. Keyboard users navigating the file list, sidebar, or settings controls have no visual cue of their position.

FIX: Add :focus-visible ring styles — 2px offset, 3:1 contrast. Replace blanket outline: none with a proper focus indicator.
P1

Missing empty states

Empty folders show a blank table with only the header row. Sidebar recents/favourites sections hide entirely when empty — the user may not know the feature exists.

FIX: Add "This folder is empty" row to the file table. Show section labels with "No items yet" rather than hiding the section.
P2

Wallpaper gate blocks startup

appReady() waits for wallpaper resolution before revealing the interface. A network-bound wallpaper fetch blocks the entire app — including the file system, which has no dependency on wallpaper.

FIX: Gate only on settingsLoaded(). Let the interface appear immediately; wallpaper resolves when ready.
P2

Accent color changes with theme

Light mode accent is #396cd8 (blue), dark mode is #24c8db (cyan). The brand hue shifts rather than maintaining identity across themes.

FIX: Pick one accent hue and derive light/dark variants from it. Keep perceptual identity consistent.
P2

Aggressive sidebar truncation

Sidebar fixed at 180px truncates long drive names and paths. Hover title attribute not accessible on touch devices.

FIX: Allow wider sidebar or add a touch-accessible tooltip. Consider resizable sidebar.
