WATCH

File rows not keyboard-focusable

File table rows use click handlers on <tr> elements with no tabindex. Tab navigation skips over every row. Users relying on keyboard navigation cannot select files or trigger context menus without tabbing past the table entirely.

FIX: Add tabindex="0" to .file-row elements. Handle Enter/Space for selection and open. Add a role="listbox" or custom grid pattern for screen reader context.
WATCH

No reduced-motion support

Loading spinner, skeleton shimmer, graph node pulse animation, and progress bar indeterminate animation all run unconditionally. Users with vestibular disorders see no way to disable motion.

FIX: Wrap all keyframe animations and transitions in @media (prefers-reduced-motion: no-preference). Provide a UI setting override in the Behavior section.
WATCH

RTL not supported

Layout uses left/right for positioning (context menu, graph tooltip, sidebar tooltip, modal, progress panel). Icons with reading direction (breadcrumb separators, back chevrons) don't mirror for RTL.

FIX: Replace left/right with inset-inline-start/inset-inline-end. Use logical margin/padding (margin-inline-end). Mirror direction-coded icons.
